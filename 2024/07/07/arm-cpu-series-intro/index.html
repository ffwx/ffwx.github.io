Cortex-X

Cortex-X 是面向未来设备的尖端 CPU

随着设备的不断发展和使用场景需要高性能 CPU，来不断提升用户体验。Arm Cortex-X 的定制计划使得定制化和差异化超越了 Arm Cortex 产品的传统路标，提供了一种为未来用例提供所需的终极性能的方法。

有如下特色:

  • 性能优先的设计

Cortex-X 有史以来最大的性能环比提升。

  • 定制化满足市场需求

通过与 Arm 的密切合作， 使项目合作伙伴能够影响产品，并针对特定市场的使用案例实现切实的改进

  • 扩展生态系统能力

与 Arm Cortex-A CPU 兼容，具有设计灵活性和可扩展性，支持 DynamIQ，可提供智能解决方案。

Arm Cortex-X925 介绍

下一代智能手机和笔记本电脑的终极性能

  • 与最新的安卓旗舰智能手机相比，单线程性能提升 36%
  • Arm 最先进、最前沿的 CPU，采用 “big.LITTLE” CPU 配置，专为提高从旗舰智能手机到笔记本电脑等设备的可扩展性而设计。
  • 行业领先的架构，具有最新的安全性和性能改进。
